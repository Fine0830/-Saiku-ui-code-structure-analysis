1、Saiku简介

OLAP（On-Line Analytical Processing，联机分析处理）是一个使分析师、管理者和执行者从原始数据中用来快速、一致、交互 
访问的一种软件技术，从而真实的反映企业的数据情况。OLAP功能特点是动态多维分析整合企业数据。
